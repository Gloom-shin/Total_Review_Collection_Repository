# 로스트파인더 프로젝트 
 - 개인프로젝트 로스트파인더를 진행하면서 느꼈던 점을 모아두는 저장소

### 문서 진행사항
 - ERD 수정중 
### 코드 진행사항 
- 기본 MVC 구현 완료
- DTO 구현, Mapstruct Mapper 적용
- Spring image S3업로드
  - S3 bucket에 대한 개념과 데이터 보안 알아보는 중 👀
